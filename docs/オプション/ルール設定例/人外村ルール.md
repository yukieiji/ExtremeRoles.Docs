# 人外村ルール

[ルールをダウンロード]({{ site.baseurl }}/download/rule/neutral/option.csv){: .btn .btn-blue }

ExRの多種多様なニュートラル役職を細かい推理等も無く気ままに楽しめるルール

## 特徴
- 他のMODには無いような多種多様の「ニュートラル」役職を楽しめます
- クルー0人、インポスター1人、リベラルも一人というそれぞれが少人数の陣営のためそれぞれの目標を達成するために動き回るため、推理要素は少なめで頭を使いません
- ExRの第四陣営「リベラル」の挙動確認が出来ます
  - 「リーダー」のキルや能力に対する圧倒的なブロック能力を体感できます

## 設定済み役職

- ニュートラル役職やインポスター役職に関してはあくまで設定例です
  - 以下のような設定も面白いと思います
    - ロールアサインフィルターを使い、「クイーン」だけではなく「ジャッカル」、「ヤンデレ」、「タッカー」も含めてどれかが割り当てられる
    - ロールアサインフィルターとアサインウェイトを使い、「ヒプノティスト」がいる場合は必ず「マッドメイト」も「ヘレティック」も割り当てられず、逆にインポスターに「ヒプノティスト」以外が割り当てられた時は「マッドメイト」か「ヘレティック」が割り当てられる

### インポスター陣営
 - キャリアー
### ニュートラル陣営
 - アリス
 - タスクマスター
 - ミッショナリー
 - ジェスター
 - トトカルチョ
 - マイナー
 - イーター
 - クイーン
 - アンブレイヤー
 - アーティスト
 - モニカ
 - ラバーズ
   - 本ルールの「ラバーズ」は「ニュートラル陣営」として割り当てられるようになっています

### リベラル
 - [リーダー]({{ site.baseurl }}/docs/roles/liberal/leader)
   - 本ルールを支える最強で無敵の役職です
   - タスクを行うことで資金を得るように調整されているので実質的なタイマー役職になっています

## カスタマイズ例

- 本ルールの「リベラル」の「リーダー」は無限にタスクが追加されるのでわざとショートタスクが無いようになっています(ショートタスクばかり追加されて早く終わることを防ぐため)
  - なので、早めにゲームが終わってしまうのであれば「リーダー」のタスクで獲得できる資金の量を調整したり、ロングタスクの量を増やしたり、勝利資金を調整するのが良いと思われます
- 「アリス」に関しては勝利難易度を調整するオプションを入れても良いと思います
