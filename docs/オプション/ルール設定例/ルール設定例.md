## ルール設定例

### チュートリアルルール
- ExRを始めて遊ぶ方向けのルール
  - ランダム性を排除し、わかりやすく派手な役職を多めに採用しております(全員役職あり)
    - 上記を設定するに当たり、複雑な設定を駆使されているので、ExRの設定のクセであったり仕様もわかりやすく網羅できます

[詳細]({{ site.baseurl }}/docs/exsample/rule/tutorial){: .btn }　[ダウンロード]({{ site.baseurl }}/download/rule/tutorial/option.csv){: .btn .btn-blue }

### 推理向けルール
- ExRの看板役職である「アサシンとマーリン」を主軸としたこってりとした推理を楽しみたい方向けのルール
- 他のMODでは味わうことのできない緻密にバランス調整された、本格的な推理を楽しむことができます

[詳細]({{ site.baseurl }}/docs/exsample/rule/guess){: .btn }　[ダウンロード]({{ site.baseurl }}/download/rule/guess/option.csv){: .btn .btn-blue }


### 高度な役職割り当て設定ルール

- ExRの複雑で高度な役職アサインシステムを網羅的に理解できるルール
- 他のMODでは絶対にできないような役職の割り当てを体験することが可能です

[詳細]({{ site.baseurl }}/docs/exsample/rule/assign){: .btn }　[ダウンロード]({{ site.baseurl }}/download/rule/assign/option.csv){: .btn .btn-blue }

### 人外村ルール

- ExRの多種多様なニュートラル役職を細かい推理等も無く気ままに楽しめるルール
- 他のMODでは「ミニ」等を使った暗黙的な協定等が必要だったのですが、ExRは「リベラル」の「リーダー」を使う事でそこら辺も必要無く既存システムの上で何も考えずに気ままにプレイできるようになっております

[詳細]({{ site.baseurl }}/docs/exsample/rule/neutral){: .btn }　[ダウンロード]({{ site.baseurl }}/download/rule/neutral/option.csv){: .btn .btn-blue }

## 使い方
2. ルール(「option.csv」)をダウンロード
4. タイトル画面から「設定」 => 「MODの設定」を開く
5. 「インポート」を押す
6. ダウンローした「option.csv」を選択
7. 完了

- ExRのインストールしたAmongUsのフォルダの下(AmongUs.exeがある場所)に「option.csv」を置いた場合、ロビー等でもインポートできます

## 共通仕様
 - ゲーム開始時直ぐに「会議ボタン」を押しやすいように調整されています
   - 初期のキルボタンのクールタイムも10秒ではないようにしています
