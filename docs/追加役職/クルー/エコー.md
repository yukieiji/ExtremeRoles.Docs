# エコー

「エコーロケーション」でプレイヤーの位置を知ることができる役職

- ボタン能力「エコーロケーション」
    - ボタンを押して発動
    - 1秒かけて範囲内の対象をスキャン
    - 最も近いプレイヤーから距離で変動するアイコンで位置を可視化する
        - オプション「「エコーロケーション」で死体を検知できるか」がオンの場合、死体も対象にできる
        - オプション「プレイヤーと死体の通知を分離」がオンの場合、死体を検知したアイコンの色は赤となる

## オプション

| オプション名 | 詳細 |
| ---- | ---- |
| エコー | 役職のスポーンレート |
| スポーン数 | 何人この役職にアサインされるか |
| 別の視界設定を持つか | ゲームで設定されているクルーの視界設定と別の視界設定を持つか |
| ビジョン | 視界の広さ |
| 視界効果を受けるか | 停電等の視界効果を受けるかどうか |
| 「エコーロケーション」のクールタイム | 「エコーロケーション」のクールタイム |
| 「エコーロケーション」の使用回数 | 「エコーロケーション」の使用回数 |
| 「エコーロケーション」の範囲 | 「エコーロケーション」の範囲 |
| 「エコーロケーション」の通知時間 | 「エコーロケーション」の通知がどれくらいの時間続くか |
| 「エコーロケーション」を発動した地点の通知 | 「エコーロケーション」を発動した地点をどの陣営に通知するか |
| 「エコーロケーション」で死体を検知できるか | 「エコーロケーション」で死体を検知できるか |
| プレイヤーと死体の通知を分離 | オンの場合、検知できたプレイヤーと死体の通知を分けて表示する |
