# C.E.O

会議で吊ったときに必ず復活する役職

- 「覚醒するタスク進捗率」が0％より高い場合、**この役職は最初何もないクルー役職としてアサインされ、タスク完了数がそのオプションの進捗率以上に達することでこの役職に覚醒する**

- この役職が会議で吊られた時、5秒後にデメリット無しで自動復活する
- オプション「この役職への投票を表示するか」がオフのとき、この役職が吊られる票数の場合は票が表示される

## 特殊会議「C.E.O会議」

- オプション「特殊会議「C.E.O会議」」が有効時、C.E.Oが吊られたときに発動する
- 吊られて起こしたC.E.Oが全権限(スキップ、吊り)を持つ会議であり、通常会議の参加者が参加できるが投票権を持たない
- 吊られて起こしたC.E.O自身を選択することは出来ない

## オプション

| オプション名 | 詳細 |
| ---- | ---- |
| C.E.O | 役職のスポーンレート |
| スポーン数 | 何人この役職にアサインされるか |
| 別の視界設定を持つか | ゲームで設定されているクルーの視界設定と別の視界設定を持つか |
| ビジョン | 視界の広さ |
| 視界効果を受けるか | 停電等の視界効果を受けるかどうか |
| 覚醒するタスク進捗率 | 覚醒するタスク進捗率 |
| この役職への投票を表示するか | オフの場合、この役職への投票が消えて表示される<br>ただし、吊られる票数の場合は票が表示される |
| 特殊会議「C.E.O会議」 | オンの場合、この役職が吊られた時に特殊会議「C.E.O会議」が発生する |
