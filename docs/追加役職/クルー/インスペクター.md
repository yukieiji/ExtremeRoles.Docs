# インスペクター

オプションで設定された行動を検知するクルーメイト

- ボタン能力: 「インスペクト」
    - ボタンを押して発動
    - 発動時間中に特定の行動を行ったプレイヤーを検知する
        - 能力が発動していると全プレイヤーに、オプションで設定された"行動が監視されているという"テキストが表示される
        - 検知後、そのプレイヤーへの矢印をこの役職のプレイヤーに表示する
        - 検知中にそのプレイヤーが死亡しても矢印の表示は続く
    - 矢印は**能力発動中**のみ表示される

## オプション

| オプション名 | 詳細 |
| ---- | ---- |
| インスペクター | 役職のスポーンレート |
| スポーン数 | 何人この役職にアサインされるか |
| 別の視界設定を持つか | ゲームで設定されているクルーの視界設定と別の視界設定を持つか |
| ビジョン | 視界の広さ |
| 視界効果を受けるか | 停電等の視界効果を受けるかどうか |
| 「インスペクト」の発動時間 | 「インスペクト」の発動時間 |
| 「インスペクト」のクールタイム | 「インスペクト」のクールタイム |
| 「インスペクト」の発動可能回数 | 「インスペクト」の発動可能回数 |
| ボタン能力発動検知 | 「インスペクト」発動中にボタン能力の発動を検知するか |
| サボタージ検知 | 「インスペクト」発動中にサボタージの発動を検知するか |
| ベント検知 | 「インスペクト」発動中にベントの使用の発動を検知するか |
