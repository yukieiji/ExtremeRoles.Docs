特殊役職「リーダー」を中心とし、ドーヴとミリタントで構成される第四の陣営

## 勝利条件
1. 資金をオプション「勝利に必要な資金」以上貯めること
2. 生存者全てがリベラルになること(**過半数勝利はない**)

「リベラル」の勝利条件判定は全陣営の中で最も優先度が低く設定されているため、同時に勝利条件を満たした時別の陣営の勝利が優先される
- 例: リーダーのキルによってクルー生存勝利とリベラルの資金勝利を満たした場合 => クルーの勝利

## 役職

### 特殊役職「リーダー」
- この役職のプレイヤーの**名前の横に資金**の状況が表示される
- 詳細はリーダーの役職情報を参照

{: .important }
>
>  * この役職はリベラルが割り当てられる時、必ず1つ存在する
>
>  * この役職はどんな状況下でも「**能力の対象**」にならない
>


### ドーヴ
- タスクを持つリベラル役職を指し、「タイプ: ドーヴ」や「ドーヴ」と表記される
- 「タイプ: ドーヴ」のデフォルト役職名も「ドーヴ」である

{: .important }
>
>  * タスクは無限に追加される
>

### ミリタント
- キルボタンを持つリベラル役職を指し、「タイプ: ミリタント」や「ミリタント」と表記される
- 「タイプ: ミリタント」のデフォルト役職名も「ミリタント」である

{: .important }
>
>  * このタイプの「リベラル」が存在している場合、他の陣営の生存勝利が発生しない
>

## オプション(グローバル設定に存在)

この陣営共通の設定として、**停電などの視界効果**を受けない

|  オプション名 |  詳細  |
| ---- | ---- |
|  リベラルのロール数　最少  | リベラル役職の数の最小数 |
|  リベラルのロール数　最大  | リベラル役職の数の最大数 |
| 勝利に必要な資金 | リベラルが勝利するのに必要な資金 |
| ドーヴ　資金ボーナス/1タスク完了あたり | 「ドーヴ」全体の設定、「ドーヴ」が1タスクを完了ごとに増加する資金の量 |
| ミリタント　資金ボーナス/キル1回あたり |「ミリタント」全体の設定、「ミリタント」が1キルするごとに増加する資金の量 |
| 視界 | リベラル陣営全体の視界のサイズ |
| 全体のベント | リベラル陣営のプレイヤーがベントを使えるか |
| リーダーへのキル許可 | 他の陣営のプレイヤーが「リーダー」をキルすることができるか |
| 資金増加率上昇値(リーダーがキルされる度) | 他の陣営のプレイヤーが「リーダー」をキルするごとに加算される資金増加率 |
| リベラル陣営がリーダー単独時のキル許可 | オンの場合、リベラル陣営が「リーダー」が一人になったときに「リーダー」をキルする事ができる |
| リーダーの自動復活 | 「リーダー」がキルされた時に自動復活するか |
| リベラル陣営がリーダー単独時の自動排除 | オンの場合、リベラル陣営が「リーダー」が一人になったときに「リーダー」が自動的にキルされ、リベラル陣営の敗北が確定する(この時復活はしない) |
| リーダー　別の視界設定を持つか | 「リーダー」の設定、「リーダー」が別の視界設定を持つか |
| 視界 | 「リーダー」の設定、「リーダー」の視界設定 | 
| リーダー タスクを持つか | 「リーダー」の設定、タスクを持ち「ドーヴ」としても扱うか |
| リーダー 資金増加率上昇値/1タスク完了あたり | 「リーダー」の設定、「リーダー」が1タスク完了ごとに加算される資金増加率 |
| リーダー 資金ボーナス/1タスク完了あたり | 「リーダー」の設定、「リーダー」が1タスクを完了ごとに増加する資金の量 |
| リーダー キルボタンを持つか | 「リーダー」の設定、キルボタンを持ち「ミリタント」としても扱うか |
| リーダー 資金増加率上昇値/1キルあたり |「リーダー」の設定、「リーダー」がキルするごとに加算される資金増加率 |
| リーダー 資金ボーナス/1キルあたり | 「リーダー」設定、「リーダー」が1キルするごとに増加する資金の量 | 
| リーダー 別のキルクールを持つか | 「リーダー」の設定、別のキルクール設定を持つか |
| キルクール | 「リーダー」の設定、「リーダー」のキルクール |
| リーダー 別のキルレンジを持つか | 「リーダー」の設定、別のキルレンジ設定を持つか |
| キルレンジ | 「リーダー」の設定、「リーダー」のキルレンジ |
| ミリタント ロール数　最小 | リベラル役職の中で「ミリタント」役職の最小数 |
| ミリタント ロール数　最大 | リベラル役職の中で「ミリタント」役職の最大数 |
| ミリタント 別のキルクールを持つか | 「ミリタント」全体の設定、別のキルクール設定を持つか |
| キルクール | 「ミリタント」全体の設定、「ミリタント」のキルクール |
| ミリタント 別のキルレンジを持つか| 「ミリタント」の設定、別のキルレンジ設定を持つか |
| キルレンジ | 「ミリタント」全体の設定、「ミリタント」のキルレンジ |
