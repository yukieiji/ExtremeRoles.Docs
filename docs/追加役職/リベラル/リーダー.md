# リーダー

- タイプ : ドーヴ/ミリタント(設定による)
    - タスクを持つ場合は「ドーヴ」
    - キルボタンを持つ場合は「ミリタント」としも扱われる


{: .important }
>
>  * この役職はリベラルが割り当てられる時、必ず1つ存在する
>
>  * この役職はどんな状況下でも「**能力の対象**」にならない
>


- 名前の横に**現在の資金**の状況が表示される
- この役職は「リーダーへのキル許可」がオフの場合、「リーダー」に対する全てのキルが無効化される
    - キルが無効化されていると、**「リーダー」への投票も全て無効票**となる
    - 以下の設定などを行うと挙動が変化する(これらの設定を行わなければ、基本的に無敵である)
        - オプション「リベラル陣営がリーダー単独時のキル許可」がオンの時、ベラル陣営が「リーダー」一人になったときに「リーダー」をキルすることが可能になる
        - オプション「リベラル陣営がリーダー単独時の自動排除」がオンの時、リベラル陣営が「リーダー」一人になったときに自動的に死亡する。この時、自動復活は行われない
    - キルが無効化されている場合かつ「ミリタント」として扱う場合、生存者計算時のみ「ミリタント」ではない(キルボタン持ちのリベラル以外)と扱われる
        - 例: 生存者がクルー陣営と「リーダー」のみになった場合 => **「ミリタント」が0人となり、クルーの勝利**

- オプション「資金増加率上昇値」はリーダーが特定の状況になった場合にリベラル陣営全員の資金が増加するときの倍率が永続増加する
    - 例:「ミリタント　資金増加率上昇値/1キルあたり」: 5、「リーダー　資金増加率上昇値/1キルあたり」: 10、「リーダー　資金ボーナス/1キルあたり」 : 10％
        1. リーダーがキル始めて資金増加率が10％増加し、資金が11(10 * 1.1)増加する。
        2. その後、リベラルのミリタントがキルすると資金が5.5(5×1.1)増加する


## オプション(グローバル設定に存在、リーダーに関連するオプションを抜粋)

|  オプション名 |  詳細  |
| ---- | ---- |
| リーダーへのキル許可 | 他の陣営のプレイヤーが「リーダー」をキルすることができるか |
| 資金増加率上昇値(リーダーがキルされる度) | 他の陣営のプレイヤーが「リーダー」をキルするごとに加算される資金増加率 |
| リベラル陣営がリーダー単独時のキル許可 | オンの場合、リベラル陣営が「リーダー」が一人になったときに「リーダー」をキルする事ができる |
| リーダーの自動復活 | 「リーダー」がキルされた時に自動復活するか |
| リベラル陣営がリーダー単独時の自動排除 | オンの場合、リベラル陣営が「リーダー」が一人になったときに「リーダー」が自動的にキルされ、リベラル陣営の敗北が確定する(この時復活はしない) |
| リーダー　別の視界設定を持つか | 「リーダー」が別の視界設定を持つか |
| 視界 | 「リーダー」の視界設定 | 
| リーダー タスクを持つか | タスクを持ち「ドーヴ」としても扱うか |
| リーダー 資金増加率上昇値/1タスク完了あたり | 「リーダー」が1タスク完了ごとに加算される資金増加率 |
| リーダー 資金ボーナス/1タスク完了あたり | 「リーダー」が1タスクを完了ごとに増加する資金の量 |
| リーダー キルボタンを持つか | キルボタンを持ち「ミリタント」としても扱うか |
| リーダー 資金増加率上昇値/1キルあたり |「リーダー」がキルするごとに加算される資金増加率 |
| リーダー 資金ボーナス/1キルあたり | 「リーダー」設定、「リーダー」が1キルするごとに増加する資金の量 | 
| リーダー 別のキルクールを持つか | 別のキルクール設定を持つか |
| キルクール | 「リーダー」のキルクール |
| リーダー 別のキルレンジを持つか | 別のキルレンジ設定を持つか |
| キルレンジ | 「リーダー」のキルレンジ |
